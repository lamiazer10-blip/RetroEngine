# Building an APK for RetroEngine

This guide explains how to build an Android APK from the React Native RetroEngine project.

## Prerequisites

- **Node.js 14+** - [Download](https://nodejs.org/)
- **Java Development Kit (JDK) 11+** - [Download](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)
- **Android SDK** - Install via Android Studio or command line
- **React Native CLI** - `npm install -g react-native-cli`
